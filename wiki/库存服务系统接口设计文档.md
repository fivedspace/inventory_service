### 类型(commodity_type)

#### 增加类型

POST：/type

> 接口参数

```
{
	type: "类型名称"
}
```

> 返回数据

```
{
	type_id: 类型id,
	type: "类型名称"
}
```

#### 查询所有类型

GET：/type

> 接口参数

```
无
```

> 返回数据

```
[
    {
        type_id: 类型id,
        type: "类型名称"
    },
    {
        type_id: 类型id,
        type: "类型名称"
    }
]
```

#### 修改指定类型

PATCH：/type

> 接口参数

```
{
	type_id: 类型id,
	type:"类型"
}
```

> 返回数据

```
{
	type_id: 类型id,
	type: "类型名称"
}
```

#### 删除类型

DELETE：/type?type_id={type_id}

> 接口参数

```
type_id: 类型id
```

> 返回数据

```
{
	message: "success"
}
```

### 商品属性类别（spec）

#### 添加商品属性类别

POST：/spec

>接口参数

```
{
	spec：[
		{
			spec_name:“商品属性类别名称”,
			field_type_id:属性值数据类型id
		},
		{
			spec_name:“商品属性类别名称”,
			field_type_id:属性值数据类型id
		}
	]
}
```

> 返回数据

```
[
	{
		spec_id:商品属性类别id,
		spec_name:"商品属性类别名称",
		field_type:"属性值数据类型"
	},
	{
		spec_id:商品属性类别id,
		spec_name:"商品属性类别名称",
		field_type:"属性值数据类型"
	}
]
```

#### 查询所有属性类别

GET：/spec

> 接口参数

```
无
```

> 返回数据

```
[
	{
		spec_id:商品属性类别id,
		spec_name:"商品属性类别名称",
		field_type:"属性值数据类型"
	},
	{
		spec_id:商品属性类别id,
		spec_name:"商品属性类别名称",
		field_type:"属性值数据类型"
	}
]
```

#### 修改属性类别

UPDATE：/spec

> 接口参数

```
{
	spec_id:商品属性类别id,
	spec_name:"商品属性类别名称"
	field_type_id:属性值数据类型id
}
```

> 返回数据

```
{
	spec_id:商品属性类别id,
	spec_name:"商品属性类别名称",
	field_type:"属性值数据类型"
}
```
