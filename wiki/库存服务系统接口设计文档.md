### 商品类型（type）

***

#### 增加商品类型

POST：/type

> 接口参数

```
{
	"type": "商品类型名称"
}
```

> 返回数据

```
{
	"type_id": 商品类型id,
	"type": "商品类型名称"
}
```

#### 查询所有商品类型

GET：/type

> 接口参数

```
无
```

> 返回数据

```
{
	"type_list":[
		{
			"type_id": 商品类型id,
			"type": "商品类型名称"
		},
		{
			"type_id": 商品类型id,
			"type": "商品类型名称"
		}
	]
}
```

#### 修改指定商品类型

PATCH：/type

> 接口参数

```
{
	"type_id": 商品类型id,
	"type":"商品类型"
}
```

> 返回数据

```
{
	"type_id": 商品类型id,
	"type": "商品类型名称"
}
```

#### 删除商品类型

DELETE：/type/{type_id}

> 接口参数

```
type_id: 商品类型id
```

> 返回数据

```
{
	"message": "success"
}
```

### 规格（spec）

***

#### 添加规格

POST：/spec

>接口参数

```
{
	"spec"：[
		{
			"spec_name":"规格名称",
			"data_type_id":规格值数据类型id,
			"spec_remark":"规格的描述"
		},
		{
			"spec_name":"规格名称",
			"data_type_id":规格值数据类型id,
			"spec_remark":"规格的描述"
		}
	]
}
```

> 返回数据

```
[
	{
		"spec_id":规格id,
		"spec_name":"规格名称",
		"data_type":"规格值数据类型",
		"spec_remark":"规格的描述"
	},
	{
		"spec_id":规格id,
		"spec_name":"规格名称",
		"data_type":"规格值数据类型",
		"spec_remark":"规格的描述"
	}
]
```

#### 查询所有规格

GET：/spec

> 接口参数

```
无
```

> 返回数据

```
[
	{
		"spec_id":规格id,
		"spec_name":"规格名称",
		"data_type":"规格值数据类型",
		"spec_remark":"规格的描述"
	},
	{
		"spec_id":规格id,
		"spec_name":"规格名称",
		"data_type":"规格值数据类型"
		"spec_remark":"规格的描述"
	}
]
```

#### 修改规格

UPDATE：/spec

> 接口参数

```
{
	"spec_id":规格id,
	"spec_name":"规格名称",
	"data_type_id":规格值数据类型id,
	"spec_remark":"规格的描述"
}
```

> 返回数据

```
{
	"spec_id":规格id,
	"spec_name":"规格名称",
	"data_type":"规格值数据类型",
	"spec_remark":"规格的描述"
}
```
