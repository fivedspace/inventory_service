# 库存服务系统接口设计文档



### 系统(system)

#### 查询所有系统信息

GET：/system

> 查询所有系统信息

```
接口参数：无
返回数据：
[{
	id:系统id,
	system_name："系统名称",
	phone:"电话",
	admin: "该系统管理员名称",
	remark: "备注"
},
{
	id:系统id,
	system_name："系统名称",
	admin: "该系统管理员名称",
	phone:"电话",
	remark: "备注"
}]
```

#### 查询指定系统信息

GET：/system?system_id={system_id}

> 查询指定系统信息

```
接口参数：system_id: 系统ID
返回数据：
{
	id:系统id,
	system_name："系统名称",
	admin: "该系统管理员名称",
	phone:"电话",
	remark: "备注"
}
```

#### 新建系统

```
	remark: "备注"	
}

	system_name："系统名称",
```


### 商品(commodity)

#### 查询指定系统的所有商品信息

GET：/commodity?system_id={system_id}

> 查询指定系统的所有商品信息

```
接口参数：system_id：系统id
返回数据：
[{
	id:商品id,
	commodity_name: "商品名称",
	quantity_in_stock: "库存剩余数量",
	type: "类型",
	system_id: "所属系统",
	remark: "备注"
},
{
	id:商品id,
	commodity_name: "商品名称",
	quantity_in_stock: "库存剩余数量",
	type: "类型",
	system_id: "所属系统",
	remark: "备注"
}]
```

#### 查询指定商品信息

GET：/commodity?commodity_id={commodity_id}

> 查询指定商品信息

```
接口参数：commodity_id:商品id
返回数据：
{
	id:商品id,
	commodity_name: "商品名称",
	quantity_in_stock: "库存剩余数量",
	type: "类型",
	system_id: "所属系统",
	remark: "备注",
	picture:[
		{
			picture_name:"图片名称",
			picture_file:img
		},
		{
			picture_name:"图片名称",
			picture_file:img
		}
	]
}
```



#### 新增商品

POST：/commodity

> 新增商品

```
接口参数：
{
	commodity_name: "商品名称",
	quantity_in_stock: "商品库存数量",
	type: "商品类型",
	system_id: "所属系统",
	remark: "备注",
	picture:[image1, image2]
}
返回数据：
{	
	id: "商品id"
	commodity_name: "商品名称",
	quantity_in_stock: "库存剩余数量",
	type: "类型",
	system_id: "所属系统",
	remark: "备注",
	picture:[
		{
			picture_name:"图片名称",
			picture_file:img
		},
		{
			picture_name:"图片名称",
			picture_file:img
		}
	]
}

```

### 类型(commodity_type)

#### 增加商品类型

POST：/type

```
接口参数:
{
	type: "类型名称"
}
返回参数:
{
	id: 类型id
    type: '类型名称'
}
```

#### 查询所有商品类型

GET：/type

```
接口参数: 无
返回参数:
[
    {
        id: 类型id
        type: '类型名称'
    },
    {
        id: 类型id
        type: '类型名称'
    }
]
```

#### 修改指定商品类型

PATCH：/type?type_id={type_id}

```
接口参数: type_id: 类型id
返回参数:
{
	id: 类型id
	type: '类型名称'
}
```

#### 删除指定商品类型

DELETE：/type?type_id={type_id}

```
接口参数: type_id: 类型id
返回参数:
{
	message: "success"
}
```
